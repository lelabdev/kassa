# Assets

Static assets for the Kassa application.

## Contents

- **Favicon**: Browser tab icon
- **Icons**: App icons and image assets
- **Logos**: Brand logos
- **Images**: UI images and illustrations

## Guidelines

- **Optimization**: Images should be optimized for web (compressed, appropriate format)
- **Accessibility**: Include alt text for all images in markup
- **Naming**: Use descriptive, kebab-case names
- **Formats**: Use modern formats (WebP, SVG) when possible
- **Size**: Keep asset sizes minimal for PWA performance

## PWA Icons

For progressive web app installation, include:
- `favicon.ico` (16x16, 32x32)
- `icon-192.png` (192x192, PNG)
- `icon-512.png` (512x512, PNG)

See `public/manifest.json` for PWA icon configuration.
